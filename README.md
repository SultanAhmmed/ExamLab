# ExamLab – Online Examination Portal

ExamLab is a modern online examination platform built with PHP, MySQL, and Bootstrap 5. It provides role-based access for students, teachers, and administrators.

## Features
- **Student**: Take exams, view results, join discussions.
- **Teacher**: Create/delete exams, guide students.
- **Admin**: Manage users, monitor activities, maintain system.
- **Leaderboard**: Top students ranked by CGPA/score.
- **Search**: Demo search for students.

## File Structure
See the provided directory structure.

## Installation
1. Clone the repository into your web server (e.g., `htdocs` for XAMPP).
2. Import `database/online_exam.sql` into your MySQL database.
3. Update `config/database.php` with your database credentials.
4. Run the project from `index.php`.

## Technologies
- PHP (no framework)
- MySQL
- Bootstrap 5
- Custom CSS & JavaScript
